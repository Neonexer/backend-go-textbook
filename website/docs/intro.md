# Добро пожаловать в «Бэкенд на Go»

Этот учебник проведёт тебя от первого HTTP-сервера на Go до полноценного бэкенда YouTube-клона в микросервисной архитектуре.

## Для кого этот учебник

Ты уже программируешь на Python, Java, JavaScript, C# или PHP и хочешь освоить Go для бэкенд-разработки. Синтаксис Go ты уже видел — этот учебник не про «что такое переменная», а про **как строить продакшен-бэкенд на Go**.

## Как устроен учебник

Три сквозных проекта по нарастающей сложности:

| Проект | Что строим | Ключевые темы |
|--------|-----------|---------------|
| **REST API для блога** | HTTP-сервер с нуля | `net/http`, `chi`, middleware, тесты, логирование |
| **Маркетплейс** | Платформа с БД и пользователями | PostgreSQL, GORM/pgx, JWT, RBAC, транзакции |
| **YouTube-клон** | Микросервисная видеоплатформа | gRPC, Kafka, Redis, observability, CI/CD, деплой |

Каждая глава — это конкретный шаг в проекте. В конце главы — **тест для самопроверки**.

## Как проходить

1. Читай главу, копируй примеры кода и запускай их локально
2. Пиши код проекта параллельно с чтением
3. Проходи тест в конце главы
4. Двигайся последовательно — каждая глава опирается на предыдущие

Весь код проектов доступен в [репозитории на GitHub](https://github.com/go-course/backend-go-textbook) в директории `code/`.

## Что понадобится

- **Go** 1.22+ ([установка](https://go.dev/dl/))
- **VS Code** с расширением Go (или любой редактор с поддержкой Go)
- **Docker** (для проектов 2 и 3 — будем поднимать PostgreSQL, Kafka, Redis)
